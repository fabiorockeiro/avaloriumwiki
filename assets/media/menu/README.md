# Menu GIFs

Each menu entry uses a GIF named after the article slug.

Replace any file here while keeping the same filename to update the menu animation without changing PHP config.

Internal article images should go in `assets/media/icons/<article-slug>/`.
Brand assets such as logo and global background stay in `assets/media/brand/`.
Large covers and banners stay in `assets/media/articles/` or `assets/media/promos/`.
