# Icons

The wiki uses inline SVG icons from `app/lib/helpers.php` for generic UI actions.

Use this folder for article-specific GIF/PNG visual items. Prefer one subfolder per menu article, using the article slug:

- `stones-guia-completo/` for elemental stones and rarity visuals.
- `sistema-de-craft/` for craft materials, tools and bars.
- `character-upgrades/` for upgrade NPC/items.
- `animus-mastery-soulpit/` for Soul Core, Soul Prism and mastery assets.

Keep `assets/media/menu/` only for animated icons shown in the sidebar/mobile menu.
